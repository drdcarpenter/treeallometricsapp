Welcome to the Urban Tree Allometrics App

This app is based on the equations in the Handbook of Urban Tree Allometric Equations and Size Characteristics, by J Fennel and L Fay. It allows users to estimate tree characteristics from others. For example, estimate tree height from diameter at breast height (DBH), or estimate DBH from crown radius.

The app uses the UTreeAllometrics R package, which can be installed on Codeberg.
